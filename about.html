About CTRL+LIFE

Welcome to CTRL+LIFE, where technology meets lifestyle. This 3D application showcases the essential tech equipment for your digital man cave - from gaming hardware to everyday tech companions.

This project was developed as a comprehensive coursework for the 3D Web Applications module, demonstrating advanced skills in interactive 3D web development. The application leverages Three.js, a powerful JavaScript library for creating and displaying 3D computer graphics in web browsers, along with modern web technologies to create an immersive digital experience.

Project Overview

CTRL+LIFE represents a culmination of web development skills applied to create an interactive 3D showcase application. The project demonstrates proficiency in:

  • Advanced 3D modeling and rendering techniques using Three.js
  • Interactive user interfaces with real-time 3D manipulation
  • Custom shader programming for visual effects
  • Responsive design principles across multiple device types
  • Backend integration with PHP and SQLite for data persistence
  • Game development concepts including collision detection and physics
  • Audio integration with user interaction feedback

The application features multiple interconnected components including a 3D model viewer with interactive controls, an asteroid dodging mini-game, and a contact system with database integration.

Technologies Used

  • HTML5 & CSS3
  • Bootstrap 5 for responsive layout
  • Three.js for 3D rendering
  • JavaScript for interactivity
  • Custom 3D models created in Blender
  • Model-Viewer.js for 3D model integration
  • GLTF 2.0 for model file format
  • GLSL for shader programming
  • PHP and SQL for database and authentication for user contacting

Project Goals

  • To create an immersive 3D showcase that demonstrates how modern technology enhances everyday life, particularly in personal spaces like a digital man cave.

  • To create a secure and efficient database system for user contact information.
  • To ensure the website is responsive and accessible on all devices.
  • To provide a visually appealing and user-friendly interface that enhances the user experience.

Methodology

  • Statement of Originality: This website was created as an assignment for the 3D Web Applications module.
  • 3D models: 3D were created using Blender as well as texturing and any animation details.
  • Design: The design was inspired by modern gaming aesthetics and crafted to match my accommodation setup as both a gamer and Computer Scientist. The UI follows a dark theme with neon accents to create an immersive digital environment. All 3D models are contained in a responsive Bootstrap canvas with custom Three.js lighting systems including:
    • Hemisphere ambient lighting for balanced illumination
    • Directional lights with shadow mapping for realistic depth
    • Interactive GUI controls allowing users to adjust light intensity and color
    • Custom shader-based backgrounds with animated gradients
    • Material properties configured for proper reflections and shadows
    The interface was designed with both aesthetics and usability in mind, featuring intuitive controls and clear visual feedback for all user interactions.
  • Responsive Design: The website is responsive and accessible on all devices.
  • Interactivity: The application features multiple layers of user interaction:
    • 3D Model Viewer: Users can rotate, zoom, and explore detailed 3D models with OrbitControls
    • Model Manipulation: Interactive controls for wireframe toggle, exploded view, part highlighting, and animations
    • Environment Controls: Users can change lighting conditions, colors, and scene backgrounds
    • Audio Feedback: Sound effects provide auditory feedback when selecting different models (coke bottle fizz, iPhone notification)
    • Asteroid Game: Fully interactive mini-game with keyboard controls, collision detection, and score tracking
    • Responsive Feedback: Visual indicators for loading states, selections, and interactive elements
    Each interaction was carefully designed to enhance user engagement and provide an intuitive experience across different devices.
  • Integration: The project integrates multiple technologies and media types:
    • 3D Models: Custom model-viewer.js implementation using Three.js for loading and rendering GLTF/GLB models
    • Custom Spaceship: Programmatically created using Three.js geometries with detailed components including body, wings, cockpit, and engines
    • Audio Integration: Sound effects processed in Audacity and integrated with JavaScript for interactive feedback
    • Animation System: Custom animation mixers for model parts and camera transitions
    • Shader Integration: GLSL shaders for background effects and visual enhancements
  • Implementation: I was able to implement the website using HTML, CSS, and JavaScript. Functional testing was done one chrome and tested on mobile devices with ngrok to ensure compatibility on mmobile devices was good, especially with window resizing. GitHub was used extensively for version control.
  • Advanced Implementation:
    • GLSL Shader Programming: Custom fragment and vertex shaders were developed for creating dynamic backgrounds, particle effects, and material properties
    • Asteroid Game: A fully functional mini-game featuring:
      • Custom spaceship model with detailed components
      • Physics-based movement with acceleration and momentum
      • Procedurally generated asteroids with random properties
      • Collision detection system with hitbox optimization
      • Score tracking with high score persistence
      • Keyboard controls (WASD/Arrows) with boost functionality
    • Performance Optimization: Implemented techniques like object pooling, frustum culling, and level-of-detail adjustments
    • Responsive Design: Dynamic adjustments to rendering quality and UI based on device capabilities
  • Security:Our PHP Functionality validates all fields , Show proper loading state while submitting , Display success/error messages appropriately , Store submissions in the SQLite database and handles errors gracefully. With communication with our SQLite backend hosted via XAMPP.
  • Publication and Documentation:
    • Successfully published the website using GitHub Pages with comprehensive documentation
    • Created detailed README file with installation instructions, features overview, and technical requirements
    • Implemented proper code organization and commenting for maintainability
    • Ensured cross-browser compatibility and responsive design for various devices
    • Optimized assets for web delivery with appropriate compression and loading strategies
    GitHub repository: https://github.com/ronaldossai/Web-3D-app
  • Technical Challenges and Solutions:
    • 3D Model Optimization: Reduced polygon count and texture sizes while maintaining visual quality
    • Loading Management: Implemented asynchronous loading with visual indicators to improve user experience
    • Mobile Performance: Created adaptive rendering settings based on device capabilities
    • Audio Integration: Developed a system for model-specific sound effects triggered by user interaction
    • Game Physics: Implemented custom collision detection optimized for performance
