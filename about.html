About CTRL+LIFE

Welcome to CTRL+LIFE, where technology meets lifestyle. This 3D application showcases the essential tech equipment for your digital man cave - from gaming hardware to everyday tech companions.

This project was developed using Three.js, a powerful JavaScript library for creating and displaying 3D computer graphics in web browsers. The models featured are custom-created and optimized for interactive web viewing.

Technologies Used

  • HTML5 & CSS3
  • Bootstrap 5 for responsive layout
  • Three.js for 3D rendering
  • JavaScript for interactivity
  • Custom 3D models created in Blender

Project Goals

CTRL+LIFE aims to create an immersive 3D showcase that demonstrates how modern technology enhances everyday life, particularly in personal spaces like a digital man cave.

By allowing users to interact with realistic 3D models of popular tech products, we provide an engaging way to explore and appreciate modern gadgets before purchasing.

Statement of Originality

This website was created as an assignment for the 3D Web Applications module. All 3D models were created by me using Blender. The website design and implementation are original works inspired by modern gaming aesthetics.
